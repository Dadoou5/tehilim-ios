## 5. Étapes Play Console

1. **Créer le compte développeur** (25 USD une seule fois si pas déjà fait) sur play.google.com/console
2. **Créer une nouvelle app** : nom = "Tehilim", langue par défaut = "Français (France)"
3. **Configuration > Détails de l'app** : remplir titre court, description courte, complète
4. **Configuration > Graphismes** : uploader icône 512px, feature graphic, screenshots
5. **Politique > Politique de confidentialité** : URL Netlify
6. **Contenu > Sécurité des données** : déclarer « aucune donnée collectée »
7. **Contenu > Classification du contenu** : remplir le questionnaire IARC (probablement « Tous » 3+ ou 13+)
8. **Production > Tests internes** (commencer ici plutôt qu'en production directe) :
   - Créer une release « V1.0.0 internal »
   - Uploader `app-release.aab`
   - Notes de version (voir section 6 ci-dessous)
   - Ajouter 1-2 testeurs (toi + ev. ton rav) via leur adresse Gmail
9. **Soumettre pour examen** — délai habituel 2-4 jours
10. Une fois approuvé en test interne, promouvoir vers **Production fermée** puis **Production publique**
