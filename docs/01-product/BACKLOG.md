# Backlog — User Stories

Format : `As a <user>, I want <goal>, so that <reason>` — Priorité (P0/P1/P2) — Estimation (S/M/L)

## EPIC : Lecture

- **US-01** [P0/M] As a reader, I want to read any of the 150 Tehilim, so that I can do my daily reading.
- **US-02** [P0/S] As a reader, I want to navigate prev/next inside a Tehilim, so that I can follow the order.
- **US-03** [P0/S] As a reader, I want to see the 5 books and the psalms inside, so that I can browse by structure.
- **US-04** [P0/M] As a reader, I want to mark a psalm as favorite, so that I can find it again.
- **US-05** [P0/S] As a reader, I want to resume my last reading, so that I save time.
- **US-06** [P1/M] As a reader, I want a reading position memory inside a long psalm, so that I don't lose my place.

## EPIC : Recherche

- **US-10** [P0/M] As a reader, I want to search a psalm by Arabic number, so that I find it instantly.
- **US-11** [P0/M] As a reader, I want to search a psalm by Hebrew letters (`כג`), so that I use my native language.
- **US-12** [P1/M] As a reader, I want tolerant search (`tehilim 23`, `psaume 23`), so that I am not blocked by syntax.
- **US-13** [P1/S] As a reader, I want suggestions if my input is incomplete, so that I correct quickly.
- **US-14** [P2/L] As a reader, I want to search inside text (full-text), so that I find a verse by content.

## EPIC : Traduction française

- **US-20** [P0/S] As a French reader, I want to toggle the FR translation globally, so that I always see it.
- **US-21** [P0/S] As a French reader, I want to toggle FR per screen, so that I can hide it during pure Hebrew reading.
- **US-22** [P0/S] As a reader, I want my preference saved, so that I don't reset it every time.

## EPIC : Tehilim du jour

- **US-30** [P0/M] As a daily reader, I want to see today's psalms (monthly cycle), so that I follow the custom.
- **US-31** [P0/M] As a daily reader, I want to see today's psalms (day-of-week cycle), so that I follow another custom.
- **US-32** [P0/S] As a daily reader, I want to choose my mode in settings, so that the app respects my practice.
- **US-33** [P2/L] As an advanced user, I want a configurable custom mode, so that I follow my community's specifics.

## EPIC : Cas de la vie

- **US-40** [P0/M] As a reader in a life situation, I want to browse categories, so that I find suitable psalms.
- **US-41** [P0/M] As a reader, I want to see psalms associated with a category, so that I read them in order.
- **US-42** [P1/S] As a reader, I want a short editorial note, so that I understand the context — without medical/religious promise.

## EPIC : Psaume 119

- **US-50** [P0/M] As a reader, I want to access Tehilim 119 by Hebrew letters, so that I read by section.
- **US-51** [P0/S] As a reader, I want to navigate between letters, so that I move freely.

## EPIC : Réglages

- **US-60** [P0/S] As a reader, I want to change text size, so that reading is comfortable.
- **US-61** [P0/S] As a reader, I want to switch dark / light / system mode.
- **US-62** [P0/S] As a reader, I want my settings persisted locally.

## EPIC : Accessibilité

- **US-70** [P0/M] As a low-vision user, I want VoiceOver support, so that I can use the app.
- **US-71** [P0/M] As a Dynamic Type user, I want all text to scale, so that my system size applies.
- **US-72** [P0/S] As any user, I want sufficient contrast, so that I read without strain.

## EPIC : Plateforme

- **US-80** [P0/M] As a user, I want the app to work fully offline (Hebrew corpus).
- **US-81** [P0/S] As a user, I want the app < 1.5s cold start.
- **US-82** [P1/S] As a user, I don't want any tracker / analytics without consent.

## EPIC : Chaîne de Tehilim (collaboratif) — livré V1.14

- **US-CH01** [P1/L] As a user, I want to create a chain for an intention and share it (WhatsApp link + QR), so that others can join. ✅
- **US-CH02** [P1/M] As a participant, I want to join with a display name and pick the Tehilim I commit to read, in real time, so that the 150 get covered. ✅
- **US-CH03** [P1/M] As a participant, I want a taken Tehilim to be locked, so that there is no double booking. ✅
- **US-CH04** [P1/M] As the chain owner, I want to extend the selection, assign the rest, close & distribute, and remove a participant. ✅
- **US-CH05** [P1/M] As a participant, I want push notifications about progress (thresholds, distribution, reminders). ✅
- **US-CH06** [P1/M] As a user, I want « My chains » grouped (Selection / Reading / Finished) with a live countdown and local delete. ✅
- **US-CH07** [P1/M] As a participant, I want to read my Tehilim offline once the chain is distributed. ✅

## EPIC : Lecture — additions V1.14
