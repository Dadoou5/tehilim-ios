# Direction Artistique

## Mood

Lecture monastique moderne. Chaleur du parchemin sans cliché religieux. Restraint. Crédibilité. Apaisement.

## Références

- Reeder (lecture, sobriété)
- Things 3 (clarté, élégance native iOS)
- Liturgical Press apps (respect du texte)
- iA Writer (le texte est roi)

À éviter :
- Skeuomorphisme parchemin / étoiles de David clichés.
- Couleurs saturées (rouge / or fluo).
- Animations lyriques.

## Couleur

Sépia profond `#7A4A14` comme accent principal. Fond crème en clair, brun très sombre en sombre. Pas de gradient. Pas de glow.

## Typographie

Sérif pour le texte (lecture longue). SF Pro pour l'UI. Hébreu : police lisible, traditionnelle, validée pour rendu RTL avec nikoud.

## Iconographie

SF Symbols uniquement. Le logo de l'app : symbole sobre (lettre ת stylisée + glyphe simple).

## Densité

Faible. Beaucoup d'air autour du texte. Aucune bordure inutile.

## Photographie / Illustration

Aucune en V1. Pas de stock photo. Pas d'illustration mystique.

## Sons

Aucun. Pas de musique d'ambiance. Haptiques discrets uniquement.

## Animations

Discrètes. Durées 150–250 ms. Transitions natives iOS.

## Promesses visuelles

- Le texte est l'élément principal sur 100 % des écrans de lecture.
- L'UI ne crie jamais.
- Le mode sombre est aussi soigné que le mode clair.
